# change-git-branch-cli

> Cli tool to change git branches interactively.

## Installation

With npm:

```
$ npm install -g change-git-branch-cli
```

With yarn:

```
$ yarn global add change-git-branch-cli
```

## Usage

```
$ change-branch
```
